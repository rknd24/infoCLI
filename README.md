infoCLI v2.0

「情報のノイズを極限まで削ぎ落とす」ニュース要約ツール

スマホやWebサイトに溢れる広告や「おすすめ記事」などのノイズを排除し、必要な1次情報だけを最速で取得するために開発した、ターミナル完結型のニュース要約ツールです。

主な機能

- カテゴリ別ニュース取得: 総合・経済・IT科学などのカテゴリからニュースをパース。

- AIによる要約: Gemini APIを活用し、記事の要点と「学習・開発への活用」という独自の切り口で考察を出力。

- 自動フォールバック: Gemini Proのレート制限を検知した際、自動でFlashモデルへ切り替えて処理を継続。

- 完全CUI: GUIを排し、コマンドライン上での数字選択のみで全ての操作が完結。

技術スタック

- Language: Python 3.12.7

- AI Model: Gemini 3.1 Pro / 2.5 Flash (Google AI Studio)

- Environment: Windows 11 / PowerShell

開発のこだわり

「ただ情報を表示するだけ」ではなく、自分の開発効率を最大化することを重視しました。

- 設計: 処理を「ニュース取得」「AI解析」「表示」のクラスに分割し、保守性を確保。

- セキュリティ: 環境変数（.env）によるAPIキーの隠蔽を徹底。

セットアップ

1. リポジトリをクローン

2. 環境変数にGOOGLE_API_KEYを設定（.envファイルまたはOSの環境変数）

3. `python infoCLI.py` で実行
